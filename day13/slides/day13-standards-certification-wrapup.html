Certification Exam Overview

Databricks Certified Associate Platform Administrator

Aspect Details
Duration 90 minutes
Questions 45 multiple choice
Passing Score 70%
Format Proctored, online
Validity 2 years

Exam Domain Coverage

Domain Weight Key Topics
Databricks Platform ~20% Architecture, workspaces, control/data plane
Identity & Access ~20% Users, groups, SCIM, SSO, service principals
Unity Catalog ~25% Metastore, governance, permissions, lineage
Compute ~20% Clusters, policies, SQL warehouses
Operations ~15% Jobs, monitoring, troubleshooting

High-Yield Exam Topics

  • Unity Catalog hierarchy and three-level namespace
  • USAGE privilege requirement for parent containers
  • SCIM provisioning workflow
  • Cluster policies attribute types
  • Job cluster vs All-purpose cluster differences
  • Account admin vs Workspace admin responsibilities
  • Delta Lake components and time travel

Common Exam Pitfalls

  • Confusing Catalog (container) with Metastore (top-level)
  • Forgetting USAGE is needed on parent containers
  • Mixing up Account admin and Workspace admin scope
  • Not knowing SCIM automates user provisioning
  • Forgetting job clusters are ephemeral (created/terminated per run)
  • Confusing Single User vs Shared access modes
