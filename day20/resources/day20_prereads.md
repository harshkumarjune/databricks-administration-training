## Certification Exam Tips

1. **Hands-on Practice**: Use Databricks Community Edition or trial workspace
2. **Know the UI**: Be familiar with Admin Console navigation
3. **Understand Concepts**: Focus on "why" not just "how"
4. **Time Management**: ~90 seconds per question average
5. **Read Carefully**: Watch for "BEST" vs "correct" answer wording
6. **Review Flags**: If unsure, flag and return later
