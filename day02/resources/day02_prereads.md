## Key Terminology

| Term | Definition |
|------|------------|
| **Account Console** | Central management interface for the entire Databricks account |
| **Workspace Admin Console** | Admin interface for a single workspace |
| **Account Admin** | Role with authority over all workspaces and account settings |
| **Workspace Admin** | Role with authority over a single workspace |
| **Entitlements** | Feature-level permissions assigned to users |
| **Workspace provisioning** | Process of creating a new Databricks workspace |
